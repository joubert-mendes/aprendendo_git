Comando para entrar com login do github

Comando para criar u novo projeto

Comando para copiar repositorio do git via link

Comando para fazer commit dos arquivos adicionados 
Git commit

Comando para trazer conteudo do repositorio remoto
Git pull

Comando para enviar arquivos alterados no computador local para o repositorio
